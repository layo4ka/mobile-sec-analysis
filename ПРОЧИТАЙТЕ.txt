??? Как развернуть проект mobile-sec-analysis

?? Что потребуется:
 • Visual Studio Code (https://code.visualstudio.com/) — редактор кода
 • Docker Desktop (https://www.docker.com/products/docker-desktop) — для запуска окружения

?

? Шаг 1. Установка Docker Desktop
 1. Установи Docker Desktop и запусти.
 2. При первом запуске может появиться окно регистрации — пропусти его.
 3. Появится окно с командой:

docker --license-accept


 4. Скопируй эту команду.
 5. Открой командную строку (cmd) и вставь её туда. Нажми Enter.

?

? Шаг 2. Установка и запуск VS Code
 1. Установи и открой Visual Studio Code.
 2. Создай на рабочем столе папку, например, Проект.
 3. Открой её через File ? Open Folder.

?

? Шаг 3. Клонирование репозитория

В терминале VS Code (открывается через Ctrl + ~ или меню Terminal ? New Terminal) выполни:

git clone https://github.com/layo4ka/mobile-sec-analysis.git

Жди завершения загрузки репозитория.

?

? Шаг 4. Переход в нужную папку

cd mobile-sec-analysis/mobile_security_service

?? Комментарий:
Ты перемещаешься в папку, где находятся ключевые файлы Dockerfile и docker-compose.yml. Отсюда Docker будет собирать и запускать сервисы.

?

? Шаг 5. Запуск проекта через Docker

В той же папке выполни:

docker compose --verbose up

Эта команда поднимет все необходимые контейнеры и сервисы.

?

? Шаг 6. Доступ к веб-интерфейсу
 1. Открой Docker Desktop.
 2. Найди запущенный контейнер, где указано 8000:8000.
 3. Кликни по числу (или иконке ссылки).
 4. В браузере откроется локальный веб-интерфейс проекта на порту 8000.

?

? Всё готово!
